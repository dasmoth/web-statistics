About

This website was created by Tom Oinn (@approx_eng), and is licensed under the Apache Software License 2.0. You can inspect and download the source code on the GitHub repository. The site has no server-side components, it runs entirely within your web browser. No data is ever sent anywhere, no cookies set, no tracking or anything unpleasant. Once the site has loaded you could unplug your computer from the network and it'll work just fine.

As with many open source projects, this site makes use of a number of excellent tools and libraries, in particular:

  • KnockoutJS - http://knockoutjs.com/index.html
  • jQuery - https://jquery.com/
  • Bootstrap - http://getbootstrap.com/
  • Yeoman - http://yeoman.io/
  • Bower - https://bower.io/
  • Gulp - http://gulpjs.com/
  • Kendo Open Source Core - http://www.telerik.com/kendo-ui/open-source-core

Finally, this was inspired by a conversation with Dr Nicola Smyllie (@NicolaSmyllie) who was annoyed at not being able to run some basic stats without installing expensive Windows-only software. This seemed like a sad state of affairs so I hacked something up before breakfast which did the job well enough. Another few hours turned it into what you see here.

Tom Oinn, June 28, 2016
