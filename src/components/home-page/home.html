Free Statistics for the Web

Version 1.0

This project aims to produce scientific visualisations, mostly for biological data, which run directly in the web browser. As such, it's not trying to do very large scale visualisations, but focuses more on customisation and ease of use.

Tools

The Circular Plot tool is intended to help circadian rhythm biologists quickly visualise a set of peak times on a variable length cycle. It includes a built-in histogram contruction, customisation of the cycle time, binning sizes and easy entry of data, as well as full customisation of the visualisation itself. In addition to the charting functionality it also computes the circular mean [1] and angular dispersion or r-value [2] and shows these quantities on the chart.
